Fast Page Plaintext Scraper

Input

A single link or a txt file in the same directory as scraper.exe with links on each newline. Include protocol: (https://, etc)

Output

Text files of plaintext for each site. Name of each text file is the url input.

Usage

Compile with gcc in MSYS2 MINGW64:

gcc -O2 -o scraper.exe scraper.c -lcurl -lpthread

Run the scraper:

./scraper https://example.com
./scraper urls.txt
