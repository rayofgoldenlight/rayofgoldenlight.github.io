A local chat server system.

The Vision

Needed a way to message people fast and log the data quickly. Made it simple to use and made it so that people can host multiple servers on the same WIFI. Prevent malicious activities by setting soft and hard caps for servers (server shuts with saved text file when soft cap is reached). For a large-scale network, the dream is to see many servers hosted on a network which people can join and make friends with those on the server. The client is under 500kb and the server is under 10mb compressed (~25mb uncompressed) so just download it and see if it is useful for you. Works on Windows devices.

Download Chat Client/Server Creator

How to Use

  • Host: Launch LocalChatHost.exe. Set your desired byte cap, TCP port, and an optional 4-digit PIN. Click start.
  • Client: Launch LocalChat.exe. Enter the server's UDP Port (typically TCP Port + 1) and click "Scan" to discover active servers. Select a server and connect.
⚠️ Important Data Warning:
Chat traffic is stored as a raw text file directly on your device. To prevent accidental data loss, it is highly recommended to keep LocalChatHost.exe in its own dedicated folder. Running it in a cluttered directory may cause it to overwrite existing text files.

Compilation Guide (MSYS2 MinGW-w64)

If you prefer to compile from source rather than running the pre-built executables, open your MSYS2 MinGW-w64 terminal and run the following commands.

Note: If Go is installed on Windows but your MSYS2 terminal says "command not found", you can link it by running this first (adjusting the path if your Go installation is elsewhere):

export PATH=$PATH:/c/Program\ Files/Go/bin
[1/4] Compiling Client Icons
windres client.rc -O coff -o client_res.o

[2/4] Compiling C Client (LocalChat.exe)
gcc client_discover.c client_res.o -O2 -o LocalChat.exe -lws2_32 -lgdi32 -luser32 -ladvapi32 -mwindows

[3/4] Compiling Server Icons
cd cmd/host
windres host.rc -O coff -o host.syso
cd ../..

[4/4] Compiling Go Server (LocalChatHost.exe)
go mod tidy
go build -ldflags="-s -w -H=windowsgui" -o LocalChatHost.exe ./cmd/host/
